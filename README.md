SEZN Wear is a sleek, responsive fashion brand landing page built with React and Vite, 
being made for an up and coming Lebanese Clothing brand.
Although this is for a company, the main focus is to help me understand javascript and frontend concepts and organization.

Note: Some componenets were not fully made by me but taken from a free components store: https://reactbits.dev/ and improved. 
Features(so far)

🎨 Interactive WebGL Background
Custom shader-based “Silk” animation using Three.js for a premium visual experience.

🧭 Responsive Layout
Optimized for both desktop and mobile while maintaining consistent design language.

🛍 Product Showcase Section
Grid-based featured collection with hover interactions.

🎯 Smooth Scrolling Navigation
Hero button scrolls seamlessly to collection section.

🧩 Reusable Components
Modular React components like Silk and StarBorder for easy customization.

🛒 UI Enhancements
Custom animated button borders and mobile-friendly cart icon.

🛠 Tech Stack
React (Vite),
Tailwind CSS,
Three.js,
@react-three/fiber
